# Strava::Api::V2

API for accessing the strava V2 API - (http://bit.ly/161qmXg)

## Installation

Add this line to your application's Gemfile:

    gem 'strava-api-v2'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install strava-api-v2
